Talks Button

Sobre o Projeto

O Talks Button é um projeto inovador que combina um brinquedo físico interativo com um aplicativo Android, facilitando a comunicação de pessoas com comprometimentos físicos e intelectuais severos. Através de uma interface lúdica, o projeto proporciona uma solução acessível e personalizada, promovendo a inclusão social e o desenvolvimento cognitivo.
