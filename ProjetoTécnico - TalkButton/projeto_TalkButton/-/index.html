Sobre o Projeto

O Talks Button é um projeto inovador que combina um brinquedo físico interativo com um aplicativo Android, facilitando a comunicação de pessoas com comprometimentos físicos e intelectuais severos. Através de uma interface lúdica, o projeto proporciona uma solução acessível e personalizada, promovendo a inclusão social e o desenvolvimento cognitivo.

Funcionalidades

  • Conexão via Bluetooth entre o brinquedo e o aplicativo
  • Personalização de sons e ações dos botões
  • Controle de funções do dispositivo móvel através do brinquedo
  • Interface simples e adaptável para diferentes usuários

Equipe de Desenvolvimento

O projeto Talks Button foi desenvolvido por uma equipe multidisciplinar de estudantes de Análise e Desenvolvimento de Sistemas, comprometidos em criar uma solução tecnológica acessível e eficiente para facilitar a comunicação assistiva.
